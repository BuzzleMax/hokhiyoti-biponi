# Hokhiyoti Biponi

E-commerce storefront foundation built with:
- Vite + React + TypeScript
- Tailwind CSS v4
- shadcn/ui
- Wouter (routing)
- Framer Motion (animations)
- lucide-react (icons)

## Getting started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages

This project is configured with a portable `base` so it can deploy to GitHub Pages without hardcoding a repository name.
